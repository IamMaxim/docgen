# Configuration

Site behavior is controlled by `docs-site.config.json`. See the [[getting-started|Getting Started]]
guide for a walkthrough, and [[structure|Project Structure]] for where files live.

- `siteTitle`, `description` — shown in the topbar and the home hero.
- `baseUrl` — where docs mount within the app (default `/docs`).
- `sectionLabels` — maps top-level folders to section names shown on the home page.
- `features` — toggles `diff`, `editor`, `search`, and `graph`.
