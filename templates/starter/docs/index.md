# Welcome to {{title}}

This is the homepage of your wiki. Edit `docs/index.md` to change it.

## What you can do here

- Write Markdown (`.md`) or MDsveX (`.svx`) files anywhere under `docs/`
- Cross-reference pages with `[[wiki-style]]` links
- Get a sidebar tree, full-text search (⌘K / Ctrl+K), and backlinks for free
- Render math with KaTeX, diagrams with Mermaid, and embed any Svelte component in `.svx`

Start with the [[getting-started|Getting Started]] guide, see how files are organized in
[[structure|Project Structure]], or jump to the [[configuration|Configuration reference]].
These `[[wikilinks]]` power the search index and the home-page link graph.
